# План изучения проекта

## Этап 1. Собрать и запустить

Цель: убедиться, что проект работает.

Команды:

```bash
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```

После запуска проверь:

- открывается ли окно;
- работает ли меню;
- двигается ли корабль;
- появляются ли астероиды;
- работает ли Game Over;
- сохраняется ли рекорд.

## Этап 2. Понять игровой цикл

Открой файл:

```text
src/Game.cpp
```

Найди метод:

```cpp
void Game::Run()
```

Он отвечает за главный цикл игры.

## Этап 3. Понять игрока

Открой файлы:

```text
include/Player.hpp
src/Player.cpp
```

Разбери методы:

- `Reset()` — начальное состояние игрока;
- `Update(float dt)` — движение игрока;
- `Draw()` — отрисовка корабля;
- `KeepInsideScreen()` — запрет выхода за границы окна.

## Этап 4. Понять астероиды

Открой:

```text
include/Asteroid.hpp
src/Asteroid.cpp
```

Астероид имеет:

- позицию;
- скорость;
- радиус;
- вращение.

## Этап 5. Понять столкновения

Открой:

```text
include/Utils.hpp
```

Там есть функция:

```cpp
bool CirclesCollide(Vector2 a, float radiusA, Vector2 b, float radiusB)
```

Это основа проверки столкновений.

## Этап 6. Добавить свою фичу

Для первого улучшения можно сделать одно из этих заданий:

1. Увеличить скорость игрока.
2. Изменить размер астероидов.
3. Добавить новый тип бонуса.
4. Добавить жизни игроку.
5. Добавить звук при подборе бонуса.
